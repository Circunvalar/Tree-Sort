📈 Benchmark BubbleSort

📘 Complejidad BubbleSort

BubbleSort tiene complejidad:

Mejor caso: O(n) Promedio: O(n²) Peor caso: O(n²)

Esto significa que el tiempo crece de forma cuadrática.

Si duplicas el tamaño del arreglo, el tiempo puede cuadruplicarse.

La línea de tendencia muestra este comportamiento.
